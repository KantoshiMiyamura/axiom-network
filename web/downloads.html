Desktop Wallet Features

🔒

OS Key Protection

Private keys sealed with Windows DPAPI, macOS Keychain, or Linux Secret Service. Keystore can only be decrypted on the originating device.

📦

Encrypted Cache

All cached balances, UTXOs, and transaction history encrypted with XChaCha20-Poly1305 using an HKDF-derived device key.

📋

Clipboard Protection

Addresses copied to clipboard are automatically cleared after 30 seconds. Prevents accidental exposure of wallet addresses.

✅

Address Validation

Live checksum verification on recipient addresses. Visual indicators for checksummed vs legacy addresses. Send button disabled for invalid addresses.

🔐

Air-Gapped Signing

Sign transactions offline on an air-gapped machine. Export unsigned transactions, sign with the wallet, and broadcast from a connected device.

🛠

Binary Integrity

SHA-256 hash of the running binary displayed in Settings. Verify the wallet has not been tampered with before use.

What's in the archive

One main binary at the root, helpers under tools/.

  • axiom — the only binary most users need (wallet, node, mining, status)
  • README.txt — quick-start guide
  • tools/axiom-node — direct full-node binary (advanced)
  • tools/axiom-keygen — standalone keypair generator (advanced)
  • tools/axiom-sign — offline transaction signer (advanced)
  • tools/axiom-bump-fee — replace-by-fee helper (advanced)

Verify Your Download

Always verify the SHA-256 checksum after downloading to ensure file integrity.

Linux

sha256sum axiom-v1.0.1-testnet.7-*.tar.gz

Windows PowerShell

Get-FileHash axiom-v1.0.1-testnet.7-*.zip -Algorithm SHA256 | Format-List

Quick Start

Two steps to get mining.

1

Extract the archive

# Linux
tar xzf axiom-v1.0.1-testnet.7-*.tar.gz
cd axiom-v1.0.1-testnet.7-*

# Windows: right-click the .zip and select "Extract All"
2

Run

./axiom mine

That command creates your wallet, starts a node, and begins mining on your own chain. Run ./axiom with no arguments for the quick-start menu, ./axiom status for live status, ./axiom wallet balance for your balance.

3

Optional — join a known peer

./axiom mine --peer HOST:9000

Repeat --peer ADDR to wire several known nodes. Without a peer, your node mines on its own chain.

Build from Source

Compile Axiom directly from the source code for maximum trust and customization.

ⓘ
Requires Rust 1.80+

Install Rust via rustup.rs if you don't have it.

git clone https://github.com/Ayano-X-Tech/axiom-network.git
cd axiom
cargo build --release -p axiom-cli

# Binary output: target/release/axiom-node
./target/release/axiom-node --version
