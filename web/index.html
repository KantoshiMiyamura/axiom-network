Why Axiom

Designed for a post-quantum world with engineering discipline, not marketing hype.

🛡

Quantum-Resistant Signatures

ML-DSA-87 (FIPS 204) provides NIST Category 5 security. Every transaction signature is safe against quantum computing attacks today and in the future.

⚙

Proof of Work

SHA3-256 mining with LWMA difficulty adjustment. Fair distribution through computational work, no staking cartels or validator committees.

⚡

30-Second Blocks

Fast confirmation times without sacrificing security. LWMA adjusts difficulty every block for stable intervals regardless of hashrate fluctuations.

🔒

Desktop Wallet

Native desktop wallet with OS-level key protection, encrypted local cache, clipboard security, and air-gapped signing. Built with Tauri + Rust.

★

Fair Distribution

No pre-mine, no ICO, no team allocation. Every AXM enters circulation through proof-of-work mining. Smooth reward decay from 50 AXM to 0.0001 AXM floor.

☐

Built in Rust

Memory-safe systems language with zero-cost abstractions. Modular crate architecture for consensus, networking, storage, wallet, and RPC.

Architecture

>>_

Unified CLI

Single axiom binary handles wallet management, node control, mining, and network status.

  • axiom start
  • axiom wallet create
  • axiom worker start
  • axiom rewards
  • axiom status
▦

Modular Crates

Clean separation of concerns across purpose-built Rust crates. Each module is independently testable and auditable.

  • axiom-consensus
  • axiom-crypto
  • axiom-wallet
  • axiom-node
  • axiom-rpc
🛡

Security Model

Defense in depth from cryptographic primitives through network protocol to operational deployment.

  • ML-DSA-87 (FIPS 204)
  • SHA3-256 PoW
  • Argon2id KDF
  • XChaCha20-Poly1305
  • OS-level key protection
