Releases

Release history for Axiom Network. Each release includes pre-built binaries for all supported platforms.

v1.0.1-testnet.7 — Public Testnet

Testnet
Testnet only — tokens have no monetary value. Mainnet has not launched.

What's new in this release

  • Single-binary entry point. The release archive now has one binary at the root — axiom — and tucks axiom-node, axiom-keygen, axiom-sign, and axiom-bump-fee under tools/. New users see one binary; advanced users keep everything they had before.
  • Friendly first run. Run axiom with no arguments and you get a wallet-aware quick-start menu — if a wallet exists it shows the address, path, and follow-up commands; if not, it explains what axiom mine will do and where the data lives.
  • Quick-start README.txt baked into the archive. Sectioned guide covering the literal command to run, the data-dir per OS, joining a peer, and a one-line note pointing power users at tools/.
  • Cleaner docs. The Quick Start collapses from three steps to two (extract → ./axiom mine). Direct-node-binary examples move below as advanced.

Inherited from v1.0.1-testnet.6 (local-first transition)

  • No project-operated bootstrap servers, no project-operated DNS seeds. A node started without --peer runs standalone.
  • Watchdog downgrades zero/low peer-count alerts from CRITICAL to INFO when the node is standalone by design.
  • Repeatable --peer ADDR flag on both axiom-node and the axiom mine wrapper.

Core stack

  • ML-DSA-87 (FIPS 204) post-quantum transaction signatures
  • SHA3-256 proof-of-work with LWMA difficulty retarget
  • v2 handshake with domain-separated 256-bit nonce, persistent node identity, and self-peer rejection
  • UTXO mempool with fee-rate ordering and admission rate limiting
  • HTTP + WebSocket RPC (localhost-bound by default)

Platforms

  • Windows x86_64 (.zip)
  • Linux x86_64, glibc 2.31+ (.tar.gz)
  • macOS builds are not produced for this release. macOS users can build from source with cargo build --release -p axiom-cli.

Quick start

# 1. Extract the archive (Linux example).
tar xzf axiom-v1.0.1-testnet.7-x86_64-unknown-linux-gnu.tar.gz
cd axiom-v1.0.1-testnet.7-x86_64-unknown-linux-gnu

# 2. Run.
./axiom mine

Join a known peer

./axiom mine --peer <host>:9000

Node downloads (GitHub release)

Binaries are published on the GitHub release. Always verify with SHA256SUMS + minisign before running — see Verifying releases.

⬇ GitHub Release

Desktop wallet installers are not part of this release. Use the CLI wallet (axiom-keygen / axiom).

Release Notes
